# Sistema de Gestão Financeira

Um sistema de gestão financeira simples desenvolvido com **HTML5**, **CSS3** e **JavaScript**, com o objetivo de facilitar o controle básico de finanças de forma prática, visual e intuitiva.

## 📌 Sobre o projeto

Este projeto foi criado para auxiliar no gerenciamento financeiro, permitindo uma organização mais eficiente das informações e proporcionando uma visualização clara dos dados inseridos pelo usuário.

A proposta do sistema é oferecer uma solução leve e funcional para controle financeiro básico, utilizando apenas tecnologias front-end.

## 🚀 Tecnologias utilizadas

- **HTML5** — estrutura da aplicação
- **CSS3** — estilização e responsividade da interface
- **JavaScript** — interatividade e lógica do sistema

## ⚙️ Funcionalidades

- Cadastro de informações financeiras
- Organização visual dos dados
- Interface simples e de fácil utilização
- Atualização dinâmica dos elementos na tela
- Estrutura leve e de fácil execução no navegador

## 💻 Como executar o projeto

1. Faça o download ou clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em seu navegador.

## 🎯 Objetivo

O principal objetivo deste projeto é demonstrar, de forma prática, a aplicação de **HTML5, CSS3 e JavaScript** na construção de um sistema funcional para controle financeiro simples.

Além disso, o projeto também serve como base para futuros aprimoramentos, como:

- integração com banco de dados
- autenticação de usuários
- relatórios financeiros
- armazenamento em nuvem
- gráficos e dashboards

## 📷 Interface

A interface foi desenvolvida com foco em simplicidade, usabilidade e boa experiência visual, tornando o sistema acessível e fácil de navegar.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foi possível aplicar conceitos importantes como:

- estruturação semântica com HTML5
- estilização moderna com CSS3
- manipulação do DOM com JavaScript
- organização de lógica no front-end
- criação de interfaces interativas
